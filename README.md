# High-Performance Geospatial Energy Dashboard
A full-stack TypeScript application visualizing 20,000+ wind turbines across Germany. Built to demonstrate handling large-scale spatial data with smooth frontend performance.

# Tech Stack
Frontend: React, TypeScript, OpenLayers (GIS), CSS-in-JS.

Backend: Node.js, Express, TypeScript.

Database: PostgreSQL with PostGIS extension for spatial indexing.

# Key Technical Features
Vector Clustering: Implemented custom clustering logic to handle 20k+ points at 60fps.

Spatial Analysis: Utilizes PostGIS for efficient coordinate storage and retrieval.

Responsive Dashboard: Real-time statistics calculation based on map data filtering.

Zoom-Aware UI: Seamlessly transitions between data clusters and individual asset views.
